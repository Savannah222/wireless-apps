int client_send_message ( in_addr_t  addr,
int  port,
char *  command

Sends commands to an instance of the sample application.

This function takes a user-supplied command string and send it either to the sample application listening on the local UNIX socket, or if addr is not NULL then the sample application listening on the remore TCP/IP socket at addr.

Parameters
[in]addrAddress of remote application to send command to.
[in]portPort of remote application.
[in]commandPointer to command string.
Returns
  • 0
