int proto_ecpri_owdm_send_req_get_ts ( uint8_t *  msg,
uint16_t  len,
struct sockaddr_in *  dest,
int  send_ts 
)

Send an OWDM request and optionally the tx time-stamp.

Sends a One-Way Delay Measurement request to a remote node, reads and saves the tx time-stamp for internal use, and if send_ts is non-zero sends a OWDM follow-up message to the remote node with the time-stamp.

Parameters
[in]msgPointer to OWDM request message.
[in]lenLength of message.
[in]destIP address of remote node to send to.
[in]send_tsPointer to buffer to put read data in.
Returns
  • Return value of proto_ecpri_send() for message if <= 0.
  • Return value of proto_ecpri_handle_timestamps() if < 0 or !send_ts.
  • Return value of proto_ecpri_send() for followup otherwise.
